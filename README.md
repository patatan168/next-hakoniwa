# Next.js Hakoniwa

This is a [Next.js](https://nextjs.org) project.

## DetaBase Setup

```bash
npm run db:init
```

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
