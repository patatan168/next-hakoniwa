# ターンログ仕様 (Turn Log Specification)

## 概要

箱庭諸島のターン更新処理等で発生する各島のログデータは、データベースの `turn_log` テーブル内にある `log`（公開ログ）および `secret_log`（秘密ログ）カラムに保存されます。

## カスタムタグ仕様一覧

現在、ログ内で使われるカスタムタグとその表示時の変換仕様は以下の通りです。フロントエンドの `TransFormHTML.tsx` にてこの置換・パース処理が行われます。

### 1. 座標表示

- **タグ**: `[c]文字列[/c]`
- **用途**: `(x, y)` のような座標を表す際に使用します。
- **変換後**: `<span class="text-[#a06040] font-bold">文字列</span>`
- **使用例**: `[c](1, 2)[/c]`

### 2. 島名（リンクあり）

- **タグ**: `[ui:UUID]島名[/ui]`
- **用途**: 該当の島（UUID）への「見学（sight）」リンク付きで島名を表示する際に使用します。
- **変換後**: `<a href="/sight?uuid=UUID"><span class="text-[#a06040] font-bold">島名</span></a>`
- **使用例**: `[ui:example-uuid-1234]テスト島[/ui]`

### 3. 島名（リンクなし）

- **タグ**: `[i]島名[/i]`
- **用途**: すでに放棄された島や、リンクを付与したくない場合の島名表示に使用します。
- **変換後**: `<span class="text-[#a06040] font-bold">島名</span>`
- **使用例**: `[i]無人島[/i]`

### 4. 計画名・プロジェクト名

- **タグ**: `[p]計画名[/p]`
- **用途**: ミサイル発射や農場整備など、島で実行された計画（Plan）の名称を強調する際に使用します。
- **変換後**: `<span class="text-[#d08000] font-bold">計画名</span>`
- **使用例**: `[p]農場整備[/p]`

### 5. 災害・警告・自爆など

- **タグ**: `[d]文字列[/d]`
- **用途**: 地震、津波、ミサイル着弾、食料不足などの災害や警告を表す、目立たせるべき赤文字テキストに使用します。
- **変換後**: `<span class="text-[#ff0000] font-bold">文字列</span>`
- **使用例**: `[d]巨大隕石[/d]`

### 6. その他強調（太字）

- **タグ**: `[b]文字列[/b]`
- **用途**: 地形名（町、森、海など）や獲得資金など、一般的な強調を示したい場合に使用します。
- **変換後**: `<span class="font-bold">文字列</span>`
- **使用例**: `[b]森[/b]`

## フロントエンドでのパースフロー

1. レンダリング対象の文字列（`secret_log` または `log`）を受け取る。
2. 正規表現（`.replace`）を用いて、上記の独自タグを一時的に `<span class="...">` などの文字列に置換する。
3. `html-react-parser` ライブラリのパーサーに通し、ReactのDOM要素（React Node）へと安全に変換してから画面に描画する。
